## 博客终于搭起来了🎇

迫于室友[小张](https://blog.131714.xyz/)的淫威😨😨、~~色诱~~、以及**威逼**下，于2025/08/17开始正式搭建这个博客。

其实很早就想搭一个自己的博客了，但一直拖延。这次终于下定决心，从零开始搞了起来。

| 项目 | 选择                                            | 理由                   |
| ---- | ----------------------------------------------- | ---------------------- |
| 框架 | [Hexo](https://hexo.io/zh-tw/)                     | 快速、简单、中文社区大 |
| 主题 | [anzhiyu](https://docs.anheyu.com/)                | 功能丰富、设计好看     |
| 域名 | [spaceship](https://www.spaceship.com/zh/domains/) | 价格便宜               |
| DNS  | [Cloudflare](https://dash.cloudflare.com/)         | 免费、功能强大         |
| 部署 | [Vercel](https://vercel.com/)                      | 自动部署、免费额度够用 |
