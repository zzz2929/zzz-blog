## 2. Everything

秒级文件搜索，Windows 自带搜索的完美替代品。

**核心优势**：基于 NTFS 文件系统索引，搜索速度比 Windows 自带搜索快 **几百倍**。

**下载：**

- [官网](https://www.voidtools.com/zh-cn/)
- [NAS](https://s.fnnas.net/s/68997fa8d1054deab3)
- [夸克网盘](https://pan.quark.cn/s/fac74b4342a3)

![everything](https://imgbed.904002.xyz/file/blog/post/Windows工具/文件类/everything.png)

**搭配推荐**：配合 [Windhawk](https://windhawk.net/) 的 `Better file sizes in Explorer details` 组件，可以在文件管理器中直接显示文件夹大小：

**典型场景：**

- 代码修改前后的对比
- 两个文件夹的差异对比和同步
- 合并代码冲突

**下载：**

## 5. Format Factory（格式工厂）

老牌文件格式转换工具，支持视频、音频、图片、文档的格式转换。

**常用功能：**

- 视频格式转换（MP4、AVI、MKV、MOV 等）
- 音频格式转换（MP3、FLAC、WAV、AAC 等）
- 图片格式转换（JPG、PNG、WEBP、BMP 等）
- 视频压缩、音频压缩
- GIF 制作
- 屏幕录制

**下载：**

- [官网](https://format-factory.en.softmany.com/)
- [NAS](https://s.fnnas.net/s/01c8c239da8149cfab)
- [夸克网盘](https://pan.quark.cn/s/8a00d37a5e26)

![Format Factory](https://imgbed.904002.xyz/file/blog/post/Windows工具/文件类/FormatFactory.png)

## 6. ReNamer

批量文件重命名工具，支持正则表达式、序号、日期、大小写转换等多种规则。

**典型场景：**

- 照片按日期批量重命名
- 文件名统一大小写
- 添加/删除前缀后缀
- 替换文件名中的特定字符

**下载：**

- [官网](https://www.den4b.com/products/renamer)
- [NAS](https://s.fnnas.net/s/4bc240c16055430ead)
- [夸克网盘](https://pan.quark.cn/s/69698d89df92)
